# Project-Advanced Scripting for Design

## วิธีการเปิดใช้งานเครื่องอาจารย์ (How to Run)

เพื่อให้การแสดงผลสไตล์และ Layout ต่างๆ ถูกต้องครบถ้วนตามที่ออกแบบไว้ รบกวนอาจารย์ดำเนินการตามขั้นตอนดังนี้ค่ะ:

1. **ดาวน์โหลดไฟล์:** ทำการดาวน์โหลด (Download) โฟลเดอร์โปรเจกต์ทั้งหมดลงในเครื่องคอมพิวเตอร์ค่ะ 

2. **เปิดด้วย VS Code:** นำโฟลเดอร์ที่ดาวน์โหลดมาเปิดผ่านโปรแกรม **Visual Studio Code** 

3. **การรันหน้าเว็บ:**
* ไปที่ไฟล์ชื่อ **`Home.html`** (หรือไฟล์หน้าแรกของโปรเจกต์)
* คลิกขวาที่ไฟล์แล้วเลือก **"Open with Live Server"** เพื่อดูการแสดงผลค่ะ

**หัวข้อ CSS ที่นำมาใช้:** ในโปรเจกต์นี้มีการเลือกใช้เทคนิคตามที่เรียนในวิชามากกว่า 5 หัวข้อ เช่น Flexbox , Grid , Animation , Responsive Design และ Utility-first Framework (Tailwind CSS) ค่ะ
